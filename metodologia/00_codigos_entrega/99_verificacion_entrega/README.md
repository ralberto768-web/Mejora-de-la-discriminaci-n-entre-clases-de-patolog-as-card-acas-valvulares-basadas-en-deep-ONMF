# 99 - Verificacion de entrega

Comprueba la carpeta `Codigos`.

## Solo sintaxis

```powershell
python verificar_todo.py --solo-sintaxis
```

Tambien se puede usar:

```powershell
python ejecutar.py --solo-sintaxis
```

## Verificacion completa reducida

```powershell
python verificar_todo.py --datos "..\Bases de Datos"
```

Genera un informe CSV y TXT dentro de `resultados_verificacion_entrega`.
